# MyCommunity

# 项目介绍
MyCommunity 是一个模仿知乎的简单问答网站，基于SpringBoot+MyBatis实现。
用户可以发表帖子、评论、点赞、私信、相互关注、查看用户信息等。
同时，网站拥有敏感词过滤、热门帖子排行、搜索帖子、系统消息通知等功能。

### 用到的技术
技术 | 名称 
----|----
Spring Boot | 容器+MVC框架
Spring Security | 认证和授权框架
MyBatis | ORM框架  
Elasticsearch | 搜索引擎
Redis | 分布式缓存
Thymeleaf | 模板引擎
Quartz | 定时任务支持
